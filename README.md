## ğŸš€ Project Overview
The **Medical Prescription Portal** is designed to digitize and simplify the prescription-sharing process.  
Patients often lose or misplace physical prescriptions. This application solves that problem by enabling doctors to **upload prescriptions online**, and patients can securely **download them anytime** using OTP verification.

The system includes:
- A **Doctor Dashboard** for uploading prescriptions  
- A **Patient Portal** to securely receive prescriptions  
- OTP-based authentication  
- Safe and secure file management
